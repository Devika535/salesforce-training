# salesforce-training
Welcome to my Salesforce training repository. This repository contains my daily practice work, Trailhead exercises, CRM concepts, screenshots, and hands-on activities completed during Salesforce learning sessions.

The main purpose of this repository is to maintain all learning activities in an organized and structured format week by week and day by day. Topics Covered

- Introduction to CRM
- Salesforce Basics
- Accounts, Contacts, Leads, and Opportunities
- Custom Objects
- Relationships in Salesforce
- Schema Builder
- Trailhead Hands-on Challenges
- Real-world Salesforce Mapping
 Learning Approach

In this training, I am learning:
- How companies use Salesforce CRM
- How customer data is managed
- How automation improves business processes
- How to create and customize Salesforce objects
- How to work with relationships and fields

Each folder contains:
- Daily notes
- Practical tasks
- Screenshots
- README explanations
- Outputs and activities
- This repository also includes screenshots and work completed from Salesforce Trailhead modules and hands-on challenges.
